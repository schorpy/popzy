Popzy is a lightweight popup for WordPress. Create beautiful popups to boost conversions and engage visitors.

== Description ==  
Popzy is a user-friendly popup plugin that helps you create high-converting modals for your website. Whether you need to capture emails, display announcements, or promote special offers, Popzy makes it easy.

**Features:**  
- Gutenberg popup editor 
- Mobile-friendly and responsive  
- Performance optimized  

== Installation ==  

1. Upload zip or install the plugin through the WordPress Plugins screen.  
2. Activate the plugin through the 'Plugins' screen in WordPress.  
3. Go to **Popzy > Add New Popup** to create your first popup.  
4. Configure the settings and publish your popup.
